Slippage is enforced on-chain from the vault’s slippageTolerance (getConfig above): the contract derives minAmountOut per request; callers do not pass it.

Keeper: Cadence relayer (RelayerRouter)

Transactions live under cadence/transactions/relayer/: handle_stakes.cdc, initiate_unstakes.cdc, finalize_unstakes.cdc, compound_and_sync_rate.cdc. Bridge-crossing txs scope FLOW fees via maxBridgeFlowFee (UFix64). Set optional keeperFlowCliExtra in config (for example --config-path flow.deploy.json). This page does not accept a Cadence private key.
